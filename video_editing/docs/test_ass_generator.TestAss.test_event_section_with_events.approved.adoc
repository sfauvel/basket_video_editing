= Event section with events

We generate an event section with following events:

|====
| Style | Layer | Text | Start | End

| ScoreA | 3 | 5 | 10 | 20
| ScoreB | 3 | 2 | 8 | 100
|====

The `Events` section generate is:

----
[Events]
Format: Layer, Start, End, Style, Name, MarginL, MarginR, MarginV, Effect, Text
Dialogue: 3,0:00:10.00,0:00:20.00,ScoreA,,0,0,0,,5
Dialogue: 3,0:00:08.00,0:01:40.00,ScoreB,,0,0,0,,2

Dialogue: 1,0:00:00.00,0:02:00.00,Score,,0,0,0,,-
Dialogue: 1,0:00:00.00,0:02:00.00,TeamA,,0,0,0,,SLB
Dialogue: 1,0:00:00.00,0:02:00.00,TeamB,,0,0,0,,NBH
Dialogue: 2,0:00:00.00,0:02:00.00,Quarter,,0,0,0,,2
----
